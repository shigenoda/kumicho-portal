## 1. この用紙の目的
この用紙は、組長交代時に「引き継ぎ袋の中身」と「関連する保管物」を紙で確認するための一覧である。

画面上の管理だけでは見落としや口頭依存が起きやすいため、引き継ぎ時は本紙を用いて、現物確認・手渡し確認・不足確認を行う。

---

## 2. 確認方法
各項目について、以下を確認する。

- 現物が存在するか
- 保管場所が合っているか
- 何に使うものか説明できるか
- 補足事項や注意点があるか
- 次年度へそのまま渡せる状態か

確認できたものは「確認」欄にチェックを入れる。
不足・破損・所在不明のものは「備考」欄へ記入する。

## 4. 引き継ぎ時に一緒に確認すること

### 4-1. 物理物品
- 倉庫の鍵
- 引き継ぎ袋本体
- すぐ使う紙資料
- 過去の案内文や控え

### 4-2. 情報
- 連絡先が最新か
- 現在の組長運用に必要なメモが揃っているか
- 曖昧なルールや未解決事項が紙またはポータルに残っているか

### 4-3. 現場対応情報
- 倉庫の場所
- 清掃道具の保管場所
- 水道や清掃後の片付け場所
- 紙だけでは伝わりにくい注意点
